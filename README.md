# requirement

- node v18
- pnpm

# Install

```bash
git clone git@github.com:ryota-murakami/react-experimental-workspace.git
cd react-experimental-workspace
pnpm install
```

# Usage

```bash
pnpm dev // run dev-server
pnpm generate // generate new experiment page
```
